# API MINI Project - [Swagger API - Pet Store 3](https://github.com/swagger-api/swagger-petstore)
###### Java, Junit, Postman, Docker

### Project Goal:
- to test 3 endpoints
- to test all the data in the endpoint response

### Get Requests
- GET inventory
- GET pet by ID
- GET pet by status

### Endpoints that we are specifcally testing
- POST add a pet
- UPDATE update a pet
- DELETE delete a pet
